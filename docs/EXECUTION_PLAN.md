# Execution Plan

This repository follows the MVP plan in `IMPLEMENTATION.md`.

Implemented slices:

- storage primitives and deterministic graph JSON
- strict notebook parser and docs extraction
- artifact runtime and object store
- managed subprocess execution
- REST and SSE backend
- built-in templates and checkpoints
- bundled browser shell for project inspection

Deferred polish beyond the MVP shell:

- full React + ReactFlow frontend
- richer artifact previews in the browser
- production-grade interactive editor proxy UX
